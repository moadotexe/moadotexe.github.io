Deploy a sleek site with GitHub Pages

This is a minimal, fast-loading template with responsive layout, dark/light theming, and sensible SEO tags. Edit index.html and push—Pages does the rest.

Get Started →
1

Drop it in your repo

Place this index.html at your repository root (or in /docs if using that Pages setting).

2

Commit & push

Use your preferred Git client: git add . && git commit -m "Initial site" && git push.

3

Publish

In your repo: Settings → Pages. Choose a source (branch or Actions). Your site will build automatically.
